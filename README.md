# Simple Alfastore API Viewer

## Cara jalanin lokal
1. Install Vercel CLI
```bash
npm i -g vercel
```

2. Jalankan project
```bash
vercel dev
```

3. Buka browser
```

http://localhost:3000
```

## Deploy ke Vercel
1. Upload project ini ke GitHub
2. Login ke https://vercel.com
3. Import repository GitHub
4. Deploy
